## Questions
After having reviewed our repository, our top questions might be:
1. Does this line of work seem sufficiently unique? There is a lot out there on SAEs, too much to read all of, but from what I can gather these are not standard practices for intervning with SAEs. However, many of these papers are from almost three years ago, which is a long time for things to change in AI.

2. Are there any tips for what makes LLM experiments 'rigorous'? We can of course run lots of iterations, isolate hyperparameters we are modifying, and create consistent and potentially automated methods of evaluation, but if there are other components that may be standard in this line of work I would love to know.

3. What is the ideal way that you or some other user might be able to interact with this project? I know it isn't a CLI, but given the nature of the project I am not entirely sure if interacting with it in a Jupyter notebook as an SAE object would be all that much easier.

Thanks!
